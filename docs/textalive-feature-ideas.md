# TextAlive App API仕様にもとづく機能提案（5案）

## 1. ビート連動タイポグラフィ
- 仕様根拠: 拍・再生位置・歌詞単語タイミング取得
- 内容: 拍に同期して歌詞のフォントサイズやウェイトを動的に変化させる
- 価値: 音楽リズムを視覚的に体験しやすい

## 2. フレーズ境界エフェクト
- 仕様根拠: フレーズ単位イベント、歌詞テキスト解析
- 内容: フレーズ開始時に背景図形を生成し、フレーズ終端で収束させる
- 価値: 楽曲構造の理解と没入感を高める

## 3. 感情推定カラーシフト
- 仕様根拠: 歌詞文字列取得、再生中イベント
- 内容: AIで歌詞感情を推定し、楽曲進行に合わせて配色パレットを遷移
- 価値: 歌詞意味と視覚演出を連動できる

## 4. サビ自動ハイライト
- 仕様根拠: セクション/繰り返し構造の取得、再生時間イベント
- 内容: サビ検出区間で図形密度・アニメーション速度・文字発光を強化
- 価値: 楽曲の山場を直感的に伝える

## 5. インタラクティブ生成キャンバス
- 仕様根拠: 再生制御（再生/一時停止/シーク）と歌詞タイミング
- 内容: ユーザー操作（クリック/ドラッグ）を受けて、現在の歌詞に紐づく図形をレイヤー生成
- 価値: 受動視聴ではなく参加型体験を実現
